## 🎯 Business & Ziel

### Was macht VENDORi GmbH?
VENDORi GmbH hilft **Marken bei der Entwicklung ihrer Online-Präsenz**.

### Zielgruppe
- **Primär**: Entscheidungsträger (B2B)
- **Segment**: Unternehmen, die ihre Online-Präsenz aufbauen/verbessern wollen
- **Ansprache**: Professionell, seriös, businessorientiert

### Hauptziel der Website
**Lead-Generierung** → Kontakt aufbauen mit potenziellen Kunden

**KPIs**:
- Kontaktanfragen
- Newsletter-Anmeldungen (optional)
- Terminbuchungen (optional)

## 📐 Technische Specs

### Tech Stack
- **Framework**: Next.js 14+ (App Router, SSR/SSG)
- **Language**: TypeScript 5 (strict mode)
- **Styling**: Tailwind CSS 3 (Mobile-First)
- **Code Quality**: ESLint + Prettier

### Wichtige Anforderungen
- ✅ **Mobile-First**: Primär für Mobile optimiert
- ✅ **DSGVO-konform**: Cookie Consent, Datenschutz, Impressum
- ✅ **Performance**: Schnelle Ladezeiten, Core Web Vitals
- ✅ **SEO/AEO**: Suchmaschinen-optimiert (Phase 2)
- ✅ **Accessibility**: WCAG 2.1 AA konform

## 📄 Content & Struktur

### Scope (Phase 1)
**Nur Homepage** - Alle Sections auf einer Seite

### Geplante Sections
1. **Sticky Header**
   - Logo
   - Navigation (minimal)
   - CTA Button (Kontakt)

2. **Hero Section**
   - Headline (Value Proposition)
   - Subheadline
   - CTA (Kontaktformular / Termin buchen)
   - Hero Visual (Bild/Illustration)

3. **About Section**
   - Wer wir sind
   - Was wir machen
   - Warum VENDORi

4. **Services Section**
   - Hauptleistungen
   - Nutzen für Kunden
   - Differenzierung

5. **Contact Section**
   - Lead-Capture Form
   - Kontaktinformationen
   - Optional: Karte/Standort

6. **Footer**
   - Links zu Datenschutz/Impressum
   - Kontaktdaten
   - Social Media (falls vorhanden)

- **Texte**: Platzhalter OK (Content ist zweitrangig)
